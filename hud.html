Help & Instructions

Character Creation

To create a new character:

  1. Choose Create New Character and pick a universe
  2. Follow the blue step bar: use Next » after each step
  3. Click portraits and cards to select gender, species, and class
  4. Use Save Progress anytime; click Finish on the last step

Editing Your Character

To edit an existing character:

  1. Select your character from the Character dropdown
  2. Make your changes in the Character Summary panel
  3. Click "Save Character" to save your changes
  4. If you try to switch characters or leave with unsaved changes, you'll be warned

Step Guide

The step guide shows your progress through character creation:

  • ○ = Step not completed
  • ● = Current step
  • ✓ = Step completed

You can click on any step to jump to that section.

Saving Your Character

The "Save Character" button is always visible in the step guide panel. It will:

  • Be disabled when there are no changes to save
  • Be enabled when you make gameplay changes (name, stats, species, class, etc.)
  • Save all your changes to the server when clicked

Character Selection

Use the Character dropdown at the top to:

  • Switch between your characters (grouped by universe)
  • Create a new character
  • See which character is currently active

Tabs

  • Character: Overview and management of your character
  • Stats: Detailed view and editing of your 20 attribute stats
  • Classes: Browse and select your character's class
  • Inventory: Use the HUD I button in-world (not managed in this Setup screen)
  • Options: Game mode settings and quick actions
  • Help: This page
